    add single fonts

    @mixin add single fonts($font: 'open-sans-regular', $fonts: null, $for-all-fonts-reference: false) { ... }

    Parameters

    parameter Nameparameter Descriptionparameter Typeparameter Default value
    $font

    represents the param which is intended to be picked from the $fonts-map

    		Variable'open-sans-regular'
    $fonts

    is the param that is meant to be changed ONLY when using this mixin inside add-all-fonts

    		Mapnull
    $for-all-fonts-reference

    this serves as a switch when using this mixin inside add-all-fonts

    		Booleanfalse

    Example

    THIS IS THE DEFAULT
  @include add-single-font;
        outputs:
            @font-face {
                 font-display: swap;
                 font-family: "open-sans-regular";
                 font-style: normal;
                 font-weight: normal;
                 src: url("../resources/fonts/open-sans-semibold.ttf");
             }
THIS IS RANDOM BUT NESTED INSIDE VALID FONT-CONFIG-MAP
  @include add-single-fonts('open-sans-semibold');
        outputs:
            @font-face {
                 font-display: swap;
                 font-family: "open-sans-semibold";
                 font-style: normal;
                 font-weight: normal;
                 src: url("../resources/fonts/open-sans-semibold.ttf");
             }
